# blkdiag
Tools to perform diagnostics on storage devices

## Usage

Refer to the integrated help for usage.  
`blkdiag --help`

## Local development setup

```sh
git clone https://github.com/GeoffreyCoulaud/blkdiag.git
cd blkdiag
python3 -m venv .venv
source .venv/bin/activate
pip install -e .
blkdiag
```
